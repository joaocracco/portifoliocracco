Experiência Criativa

Primeiro Projeto - Jogo no Construct 3

O primeiro projeto realizado na disciplina "Experiência Criativa" foi um jogo desenvolvido no Construct 3, criado em colaboração com meus colegas durante o primeiro mes de aula. Inicialmente, tivemos diversas ideias para o jogo, mas a aparência final do projeto foi algo mais baseado em um modelo pronto do site nos permetindo um maior contole de eventos, com uma interface bem trabalhada. O objetivo do jogo é desafiar o jogador com puzzles e obstáculos, oferecendo uma experiência envolvente.

Foi uma excelente oportunidade de dar entrada aos conceitos de programação e design de jogos, além de aprimorar minhas habilidades em trabalho em equipe. O jogo foi desenvolvido em apenas um mês, o que exigiu uma boa organização e colaboração entre os membros da equipe.

Segundo Projeto - Site Educacional sobre a Tabela Periódica

O segundo projeto realizado na disciplina "Experiência Criativa" foi um site educacional criado para ajudar adolescentes do fim do ensino fundamental a aprenderem sobre a tabela periódica de maneira prática e interativa. Utilizando o Processing em Java, o site tinha como objetivo não apenas ensinar sobre os elementos da tabela, mas também proporcionar uma forma divertida e acessível de explorar a química.

A principal funcionalidade do site era apresentar uma lista de elementos químicos com exemplos práticos e breves descrições sobre suas propriedades e aplicações no cotidiano. Além disso, criamos uma página específica direcionada a exercícios interativos que permitiam aos usuários testar seus conhecimentos sobre a tabela periódica de forma dinâmica.

O projeto foi desenvolvido de forma colaborativa com meus colegas, o que tornou o processo de criação ainda mais enriquecedor. Trabalhar com o Processing em Java foi uma grande oportunidade de aprender uma nova linguagem de programação e entender como ela pode ser aplicada em projetos educacionais. Esse projeto não só fortaleceu meu conhecimento em Java, mas também me ensinou a importância da colaboração e da comunicação em equipe para alcançar os objetivos de forma eficaz.

Ao final do desenvolvimento, tivemos a satisfação de ver como o site foi capaz de simplificar e tornar mais interessante o aprendizado da tabela periódica, proporcionando uma experiência prática e interativa para os estudantes.

Terceiro Projeto - Site Educacional para Crianças

O terceiro projeto que desenvolvi foi um site educacional voltado para crianças, criado em HTML, e baseado em um projeto anterior. Este site tem como principal objetivo proporcionar um aprendizado de inglês de forma lúdica, envolvente e intuitiva, utilizando conceitos básicos de inglês, matemática, cores, números, família e animais.

Pensado para ser uma plataforma divertida e acessível, o site oferece uma variedade de exercícios interativos que estimulam as crianças a aprenderem e praticarem o vocabulário e conceitos de forma prática. Através de atividades simples e jogos educativos, as crianças podem explorar e aprender vocabulários do cotidiano, como os nomes das cores, animais e números, além de noções de matemática básica.

O projeto visa não apenas ensinar inglês, mas também incentivar o pensamento lógico e a resolução de problemas de maneira divertida e educativa. Utilizando conceitos de matemática de forma contextualizada, o site oferece atividades que ajudam as crianças a entenderem noções como contagem, formas geométricas e até a estrutura básica de frases em inglês.

Como parte do desenvolvimento desse site, tive a oportunidade de explorar mais sobre HTML e sobre como a construção de uma interface simples, porém interativa, pode fazer uma grande diferença na experiência de aprendizado de uma criança.

Raciocínio Algorítmico

Primeiro Projeto - Jogo de Pedra, Papel e Tesoura

Desenvolvi um jogo simples de Pedra, Papel e Tesoura com o objetivo de aprimorar minhas habilidades em lógica de programação e interatividade com o usuário. O jogo foi desenvolvido em Python e possibilita que o jogador escolha entre uma das três opções e o computador faça sua escolha aleatoriamente.

Segundo Projeto - Máquina de Vendas

O projeto da máquina de vendas foi desenvolvido com o objetivo de automatizar o processo de compra e troco para diferentes tipos de produtos. A máquina foi programada para aceitar moedas e cédulas, calcular o valor total inserido e fornecer o troco exato.

O maior desafio foi implementar o algoritmo que calcula o troco exato em centavos, garantindo que o sistema fornecesse o troco com as menores quantidades de moedas possíveis. Para isso, utilizamos um sistema de contagem de moedas e cédulas, que foram retiradas conforme a necessidade.

Fundamentos de Sistemas ciberfisicos

Seminario wowki

Este projeto foi realizado em colaboração com meus colegas, no contexto da disciplina de Ciberfísicos. Utilizamos o conhecimento adquirido nas aulas para desenvolver um sistema de monitoramento de temperatura e umidade em tempo real, aplicando sensores em três UPAs localizadas em Curitiba. O projeto foi realizado com base no último trabalho da disciplina, onde o foco principal era coletar e analisar dados ambientais.

Para a implementação, utilizamos a linguagem Micropython, que nos permitiu trabalhar com programação de sistemas embarcados de forma eficiente. A experiência nos ajudou a entender melhor o funcionamento de sensores e a integração entre hardware e software, além de expandir nosso conhecimento sobre novas linguagens de programação e suas aplicações práticas.
